**Annual Medical Cost Prediction**

A machine learning project that predicts individual annual medical expenses based on demographic, lifestyle, and health-related features. This project demonstrates end-to-end data science workflow including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation.

**Project Overview**

Healthcare costs are influenced by multiple factors such as age, BMI, smoking habits, and region. This project builds predictive models to estimate annual medical costs, helping stakeholders make data-driven decisions.

🚀 Objectives
Perform Exploratory Data Analysis (EDA) to uncover patterns
Engineer meaningful features for improved model performance
Train and compare multiple regression models
Evaluate models using appropriate metrics
Deploy a reproducible and scalable ML pipeline

**Dataset**

The dataset typically includes the following features:

age: Age of the individual
sex: Gender
bmi: Body Mass Index
children: Number of dependents
smoker: Smoking status
region: Residential region
charges: Medical costs (target variable)
🔍 Exploratory Data Analysis (EDA)

**Key insights discovered:**

Smokers incur significantly higher medical costs
BMI and age show strong correlation with charges
Regional differences have moderate impact
